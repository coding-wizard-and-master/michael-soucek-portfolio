# Portfolio

This is a portfolio website built using Next.js, Tailwind CSS, and Vite. The project showcases a personal portfolio layout with sections for the homepage, about, projects, and contact information.

## Features

- **Responsive Design**: The website is designed to be fully responsive, adapting to various screen sizes.
- **Hero Section**: A welcoming hero section on the homepage with a call to action.
- **Project Showcase**: A dedicated page to showcase various projects with individual project cards.
- **About Section**: Information about the portfolio owner.
- **Contact Information**: A contact page with a form or contact details.

## Technologies Used

- Next.js
- Tailwind CSS
- Vite
- TypeScript

## Getting Started

To get a local copy up and running, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/portfolio-clone.git
   ```

2. Navigate to the project directory:
   ```
   cd portfolio-clone
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm run dev
   ```

5. Open your browser and go to `http://localhost:3000` to see the website in action.

## Folder Structure

```
portfolio-clone
├── src
│   ├── pages          # Contains the page components
│   ├── components     # Contains reusable components
│   ├── styles         # Contains global and Tailwind CSS styles

├── package.json       # NPM configuration file
├── tsconfig.json      # TypeScript configuration file
├── tailwind.config.js # Tailwind CSS configuration
├── postcss.config.js  # PostCSS configuration
└── vite.config.ts     # Vite configuration
```
